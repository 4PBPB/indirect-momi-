## 1. grant administraor
i. cmd 관리자 권한으로 실행

ii. `net user administrator /active:yes`

## 2. open expolorer
i. `C:\Windows\sysWOW64` 실행

ii. 숨김 항목 보이기, 파일 확장자 보이기

## 3. find the jness
i. wtf alphabet.**exe** 검색 

ii. 디지털 서명이 회사명인지 확인

iii. 해당 파일 삭제

## 4. if MayaSvcEc:

i. 작업 관리자 -> 서비스 -> Maya 검색 -> 서비스로 이동

ii. M 입력로 찾기 -> 속성 -> 시작 유형 사용 안함 -> *중지 -> reboot*


## another. disable the BitLocker
i. 제어판 실행

ii. BitLocker 드라이브 암호화

iii. 해제 후 몇십 분 대기

## another-2. safe mode boot
i. shift + 재시작

ii. 문제 해결 -> 고급 옵션 -> 시작 설정
